# Quiz App

A full-stack quiz application with a Node.js/Express backend and a vanilla HTML/CSS/JS frontend.

## Features
- **User Portal**: Register, Login, Take Quizzes, View History.
- **Admin Portal**: Manage Subjects, Quizzes, Questions, Users, and Branches.
- **Secure Auth**: JWT-based authentication and password hashing.
- **Responsive Design**: Works on mobile and desktop.

## Setup

1. **Backend**:
   ```bash
   cd backend
   npm install
   npm start
   ```

2. **Frontend**:
   - Open `frontend/index.html` in a browser or serve using a static server (e.g., Live Server).

## Deployment
- **Backend**: Deploy to Render/Railway/Heroku.
- **Frontend**: Deploy to Vercel/Netlify.
- Update `frontend/js/config.js` with your production API URL.
