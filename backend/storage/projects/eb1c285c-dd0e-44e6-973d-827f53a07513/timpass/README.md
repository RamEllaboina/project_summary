# Love Potti - Journey Into My Heart

A beautiful, interactive 3D web experience built with Three.js to celebrate love.

## Features
- **3D Interactive Heart**: A gently beating heart floating in a cosmic scene.
- **Scroll to Zoom**: Journey inside the heart to discover memories.
- **Photo Gallery Tunnel**: Fly past floating memories as you delve deeper.
- **Login Protected**: Secure entry with username and password.
- **Typing Text Reveal**: A heartfelt message awaits at the core.
- **Atmospheric Audio**: Background music enhances the emotional journey.

## Technologies Used
- HTML5, CSS3, JavaScript
- Three.js (3D Rendering)
- GSAP (Animations)
- GitHub Pages (Deployment)

## How to Run Locally
1. Clone the repository.
2. Open `romantic_heart.html` in your browser (use a local server like Live Server for best results).
3. Experience the love! ❤️
