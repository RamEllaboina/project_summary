## 🛠️ Frontend Integration

### **Step 1: Call AI Detection Service**

```javascript
// Use the dedicated AI detection endpoint
const detectAIGeneration = async (projectData) => {
    const response = await fetch('http://localhost:8003/detect', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(projectData)
    });
    
    const result = await response.json();
    const aiDetection = result.aiDetection;
    
    // Extract AI detection data
    return {
        level: aiDetection.level,           // "high" | "medium" | "low"
        score: aiDetection.score,           // 0.0 - 10.0
        confidence: aiDetection.confidence,   // 0.0 - 1.0
        probability: calculateProbability(aiDetection.level, aiDetection.score),
        reasoning: aiDetection.reasoning,
        signals: aiDetection.signals,
        suspiciousFiles: aiDetection.suspicious_files
    };
};
```

### **Step 2: Calculate AI Generation Probability**

```javascript
const calculateProbability = (level, score) => {
    if (level === 'high' || score >= 8.0) {
        return 85 + Math.min(score * 2, 15); // 85-100%
    } else if (level === 'medium' || score >= 5.0) {
        return 50 + (score * 5); // 50-75%
    } else {
        return Math.min(score * 20, 25); // 0-25%
    }
};
```

### **Step 3: Update UI**

```javascript
const updateAIDetectionUI = (aiResult) => {
    // Update probability display
    document.getElementById('ai-probability').textContent = 
        Math.round(aiResult.probability) + '%';
    
    // Update color based on level
    const colors = {
        high: '#ff4444',    // Red
        medium: '#ff9800',  // Orange  
        low: '#28a745'      // Green
    };
    document.getElementById('ai-probability').style.color = 
        colors[aiResult.level];
    
    // Update message
    document.getElementById('ai-message').textContent = 
        aiResult.reasoning;
};
```

## 🐛 Common Issues & Solutions

### **Issue: "AI Generation Probability 0%"**
**Cause**: Frontend not calling AI detection service
**Solution**: 
```javascript
// ❌ Wrong - Only calling main service
const result = await fetch('/api/evaluate', {...});

// ✅ Correct - Call AI detection service
const aiResult = await fetch('http://localhost:8003/detect', {...});
```

### **Issue: "Not giving AI detection correctly"**
**Cause**: Frontend not processing AI detection response
**Solution**:
```javascript
// ❌ Wrong - Not extracting AI detection data
const result = await response.json();
console.log(result.aiDetection); // undefined

// ✅ Correct - Extract AI detection properly
const result = await response.json();
const aiDetection = result.aiDetection;
console.log(aiDetection.level); // "high"
console.log(aiDetection.score); // 8.5
```

### **Issue: "AI detection not working"**
**Cause**: Services not running or wrong ports
**Solution**:
```bash
# Make sure both services are running
# Terminal 1:
cd ai_engine && python main.py         # Port 8002

# Terminal 2:  
cd ai_detection_service && python main.py  # Port 8003
```
