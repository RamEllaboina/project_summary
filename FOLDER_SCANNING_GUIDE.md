# AI Detection Folder Scanning Guide

## 🎯 New Feature: Dynamic Folder Scanning

The AI detection service now supports **dynamic folder scanning** for uploaded projects!

## 🚀 Features Added

### **1. Individual File Detection** (Existing)
```javascript
// Scan individual files
const response = await fetch('http://localhost:8003/detect', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        projectId: "test-project",
        importantFiles: [
            { path: "file.js", content: "// Generated by AI assistant..." }
        ]
    })
});
```

### **2. Folder Scanning** (NEW)
```javascript
// Scan entire folders dynamically
const response = await fetch('http://localhost:8003/scan-folder', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        folderPath: "/path/to/uploaded/folder",
        projectId: "folder-scan-123"
    })
});
```

## 📁 Folder Scanning Features

### **Supported File Types:**
- **JavaScript**: `.js`, `.jsx`, `.ts`, `.tsx`, `.vue`
- **Python**: `.py`
- **Java**: `.java`
- **C/C++**: `.c`, `.cpp`, `.h`
- **C#**: `.cs`
- **PHP**: `.php`
- **Ruby**: `.rb`
- **Go**: `.go`
- **Rust**: `.rs`
- **Web**: `.html`, `.css`, `.scss`, `.less`
- **Scripts**: `.sh`, `.bat`, `.ps1`
- **SQL**: `.sql`
- **Mobile**: `.dart`, `.swift`, `.kt`, `.scala`

### **Automatic Language Detection:**
The scanner automatically detects the primary language based on file extensions.

### **Smart Filtering:**
- **Ignores**: `node_modules`, `__pycache__`, `.git`, `dist`, `build`
- **Hidden Files**: Skips files starting with `.`
- **Code Only**: Only scans files with supported extensions

## 🛠️ Frontend Integration

### **1. Folder Selection**
```javascript
// Let user select folder to scan
async function selectFolderForScanning() {
    const folderInput = document.getElementById('folder-input');
    const folderPath = folderInput.files[0]?.path;
    
    if (!folderPath) {
        alert('Please select a folder');
        return;
    }
    
    // Show loading state
    showLoadingState(true);
    
    try {
        const result = await scanFolderForAI(folderPath);
        displayFolderScanResults(result);
    } catch (error) {
        console.error('Folder scan failed:', error);
        alert('Failed to scan folder: ' + error.message);
    } finally {
        showLoadingState(false);
    }
}
```

### **2. Folder Scanning Function**
```javascript
async function scanFolderForAI(folderPath) {
    const response = await fetch('http://localhost:8003/scan-folder', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            folderPath: folderPath,
            projectId: `folder-scan-${Date.now()}`
        })
    });
    
    if (!response.ok) {
        throw new Error(`HTTP ${response.status}: ${response.statusText}`);
    }
    
    return await response.json();
}
```

### **3. Display Results**
```javascript
function displayFolderScanResults(result) {
    const aiDetection = result.aiDetection || {};
    
    // Update main AI detection display
    updateAIDetectionDisplay(aiDetection);
    
    // Show folder-specific information
    const folderInfo = document.getElementById('folder-info');
    if (folderInfo) {
        folderInfo.innerHTML = `
            <div class="folder-stats">
                <h3>📁 Folder Scan Results</h3>
                <p><strong>Folder:</strong> ${result.folderPath}</p>
                <p><strong>Files Scanned:</strong> ${result.filesScanned}</p>
                <p><strong>Scan Time:</strong> ${new Date(result.scanTime * 1000).toLocaleString()}</p>
            </div>
        `;
    }
    
    // Show suspicious files
    const suspiciousFiles = aiDetection.suspicious_files || [];
    const suspiciousList = document.getElementById('suspicious-files');
    if (suspiciousList && suspiciousFiles.length > 0) {
        suspiciousList.innerHTML = `
            <h4>🚨 Suspicious Files (${suspiciousFiles.length})</h4>
            <ul class="suspicious-list">
                ${suspiciousFiles.map(file => `
                    <li class="suspicious-file">
                        <strong>${file.file}</strong>
                        <span class="issues">${file.issues.join(', ')}</span>
                    </li>
                `).join('')}
            </ul>
        `;
    }
}
```

## 🔄 Usage Examples

### **Example 1: Upload Folder**
```javascript
// Frontend folder upload handler
async function handleFolderUpload(event) {
    const files = event.target.files;
    
    if (files.length === 0) return;
    
    // Get folder path (browser-dependent)
    const folderPath = files[0].webkitRelativePath || files[0].path;
    
    // Scan the folder
    const scanResult = await scanFolderForAI(folderPath);
    
    // Display results
    displayFolderScanResults(scanResult);
}
```

### **Example 2: Backend Integration**
```javascript
// Combine with main analysis
async function analyzeProjectWithFolder(folderPath) {
    // Step 1: Scan folder for AI detection
    const aiResult = await scanFolderForAI(folderPath);
    
    // Step 2: Get main analysis
    const mainResult = await fetch('http://localhost:8002/evaluate', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            projectId: aiResult.projectId,
            language: detectLanguage(aiResult.files),
            metrics: calculateMetrics(aiResult.files),
            importantFiles: aiResult.files
        })
    });
    
    const mainData = await mainResult.json();
    
    // Step 3: Combine results
    const combinedResult = {
        ...mainData,
        aiDetection: {
            ...mainData.aiDetection,
            ...aiResult.aiDetection  // Prioritize AI detection results
        }
    };
    
    return combinedResult;
}
```
